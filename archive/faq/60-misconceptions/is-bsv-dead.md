**Short answer.** "Dead" is something you can check for yourself — so please check, rather than take anyone's word for it, including ours. The BSV Blockchain's activity is publicly visible, in real time, on viewing websites such as WhatsOnChain. There you can see its blocks — the bundles of payments the network adds continuously — and the payments flowing through it. The software is actively developed, and applications run on the network. Whether that level of use satisfies any given person is a matter of opinion. Whether the network is running is a fact you can confirm in one browser tab.

## More detail

When someone declares a blockchain "dead," they usually mean one of four very different things. It helps to pull them apart:

- **"The network stopped."** For BSV this is false, and you can check it directly. The flow of blocks and payments is public ([[who-runs-the-bsv-network]]). This is the only meaning of "dead" that is purely a matter of fact.
- **"Fewer places trade it."** Sometimes true at a given moment. Trading venues add and remove coins for business and regulatory reasons ([[why-do-exchanges-delist]], [[orange-gateway-mica]]). But that describes the shops' shelves, not the network itself ([[exchange-problems-mean-bsv-problems]]).
- **"The price is low."** This FAQ does not discuss price, as a matter of policy ([[no-price-talk]]). Either way, talk about price is not a report on whether the network is working.
- **"Nobody uses it or builds on it."** This is a question of how much. The honest way to answer it is with activity you can check — the volume of payments on the blockchain, released software ([[how-is-bsv-developed]]), and applications in operation ([[data-on-chain]]) — rather than bold claims in either direction.

There is a useful habit in all this, and it applies well beyond BSV. When you read a dramatic claim about any blockchain, ask which of these four things is actually being said. Then check the ones that can be checked ([[ecosystem-who-is-who]]).
