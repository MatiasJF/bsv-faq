**Short answer.** The BSV Association (BSVA) is a non-profit based in Switzerland that looks after the BSV Blockchain. It maintains the software the network runs on, keeps the technical rulebook, teaches people how to use the technology, and promotes it. It is a caretaker, not an owner: it does not control the network, does not run exchanges, and does not hold anyone's coins.

## More detail

A shared system like BSV needs someone to do the shared housekeeping: keep the software up to date, write down the standards, help with security, and teach newcomers. For BSV, that job belongs to the Association — a non-profit based in Zug, Switzerland.

What it does, concretely: it maintains and publishes the software that the payment-processing businesses run; it keeps the technical rulebook; it runs education programs (including the BSV Academy); it supports projects building on BSV; and it speaks for the technology with institutions and governments.

Just as important is what it is *not*: it does not run the network (independent businesses do); it is not a financial company (it holds no customer money and runs no marketplace); and it does not control the blockchain — it publishes the software, but the independent businesses that run the network choose for themselves whether to use it.

Why this matters to you: when something happens in the news — an exchange closes, an app shuts down — knowing what the Association actually does (and doesn't do) tells you what it could and couldn't have decided.
